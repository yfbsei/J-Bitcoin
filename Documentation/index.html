J-Bitcoin

A comprehensive JavaScript/TypeScript cryptocurrency wallet library focused exclusively on Bitcoin (BTC) with both custodial and non-custodial wallet support.

npm install j-bitcoin
Get Started View on GitHub

Powerful Bitcoin Development

Wallet Types

Both custodial (traditional single-key) and non-custodial (threshold signature) wallets with full TypeScript support.

HD Key Derivation BIP32

Hierarchical Deterministic wallets with BIP32/BIP39/BIP44 compliance and built-in Bitcoin constants.

Threshold Signatures TSS

Advanced multi-party signature schemes enabling distributed key management without trusted dealers.

Modern Cryptography

ECDSA signatures, Schnorr signatures (BIP340), and threshold cryptography with full verification support.

Address Formats

Legacy P2PKH addresses and SegWit Bech32 addresses with seamless conversion utilities.

TypeScript First NEW

Complete TypeScript definitions with IntelliSense, autocomplete, and compile-time error detection.

Quick Start

Basic Custodial Wallet

import { Custodial_Wallet, BIP44_CONSTANTS } from 'j-bitcoin';

// Generate new wallet
const [mnemonic, wallet] = Custodial_Wallet.fromRandom('main');
console.log('Mnemonic:', mnemonic);
console.log('Address:', wallet.address);

// Use built-in convenience methods for standard Bitcoin addresses
wallet.deriveReceivingAddress(0);  // First receiving address
wallet.deriveChangeAddress(0);     // First change address

// Get wallet summary
const summary = wallet.getSummary();
console.log(`Generated ${summary.receivingAddresses} receiving addresses`);

// Sign a message
const [signature, recoveryId] = wallet.sign("Hello Bitcoin!");
console.log('Signature valid:', wallet.verify(signature, "Hello Bitcoin!"));

Threshold Signatures (2-of-3)

import { Non_Custodial_Wallet } from 'j-bitcoin';

// Create 2-of-3 threshold wallet
const wallet = Non_Custodial_Wallet.fromRandom("main", 3, 2);

// Get shares for distribution
const shares = wallet._shares;
console.log('Distribute shares to participants:', shares);

// Generate threshold signature
const signature = wallet.sign("Multi-party transaction");
console.log('Threshold signature:', signature.serialized_sig);

// Get detailed wallet information
const summary = wallet.getSummary();
console.log(`${summary.thresholdScheme} threshold wallet - ${summary.securityLevel} security`);

Type-Safe Wallet Development

import { 
  Custodial_Wallet, 
  ECDSASignatureResult, 
  ChildKeyInfo,
  NetworkType,
  WalletSummary
} from 'j-bitcoin';

// Generate new wallet with full type safety
const network: NetworkType = 'main';
const [mnemonic, wallet]: [string, Custodial_Wallet] = 
  Custodial_Wallet.fromRandom(network);

// TypeScript knows the exact return types
const [signature, recoveryId]: ECDSASignatureResult = wallet.sign("Hello Bitcoin!");
const isValid: boolean = wallet.verify(signature, "Hello Bitcoin!");

// Full type safety for child keys
wallet.deriveReceivingAddress(0).deriveChangeAddress(0);
const childKeys: ChildKeyInfo[] = wallet.getChildKeysByType('receiving');

// Complete wallet summary with types
const summary: WalletSummary = wallet.getSummary();
console.log(summary.network);            // string
console.log(summary.derivedKeys);        // number

Advanced Threshold Signatures

import { 
  Non_Custodial_Wallet,

Custodial_Wallet Enhanced

Traditional single-party control wallet using hierarchical deterministic key derivation (BIP32) with standard ECDSA signatures. Enhanced in v2.0.0 with Bitcoin-specific utilities and convenience methods.

Static Methods

Custodial_Wallet.fromRandom(net, passphrase?)

Generates a new random wallet with cryptographically secure mnemonic phrase.

Parameters
  • net string - Network type ('main' or 'test')
  • passphrase string optional - Additional security passphrase
Returns

[string, Custodial_Wallet] - Mnemonic phrase and wallet instance

const [mnemonic, wallet] = Custodial_Wallet.fromRandom('main', 'secure-pass');
console.log('Mnemonic:', mnemonic);
console.log('Address:', wallet.address);
Custodial_Wallet.fromMnemonic(net, mnemonic, passphrase?)

Creates a wallet from an existing BIP39 mnemonic phrase with optional passphrase.

Parameters
  • net string - Network type ('main' or 'test')
  • mnemonic string - 12-word BIP39 mnemonic phrase
  • passphrase string optional - Passphrase used during generation 
const mnemonic = "abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon about";
const wallet = Custodial_Wallet.fromMnemonic('main', mnemonic, 'password');

bip39.mnemonic() BIP39

Parameters

Returns

const invalidMnemonic = "abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon";

Built-in constants for standardized Bitcoin derivation paths.

Constants Structure

BIP44_CONSTANTS

Collection of Bitcoin-specific BIP44 derivation constants.

import { BIP44_CONSTANTS } from 'j-bitcoin';

// Structure
const constants = {
  PURPOSE: 44,                    // BIP44 purpose field
  COIN_TYPES: {

generateDerivationPath NEW v2.0.0

Generate a BIP44 derivation path for Bitcoin.

Function Details

generateDerivationPath(options)

Creates a complete BIP44 derivation path from options.

Parameters
  • options Object - Path generation options
  • options.purpose number optional - BIP purpose (default: 44)
  • options.coinType number optional - Coin type (default: 0)
  • options.account number optional - Account index (default: 0)
  • options.change number optional - Change chain (default: 0)
  • options.addressIndex number optional - Address index (default: 0)
Returns

string - Complete BIP44 derivation path

import { generateDerivationPath, BIP44_CONSTANTS } from 'j-bitcoin';

// Generate standard Bitcoin receiving address path
const receivingPath = generateDerivationPath({
  purpose: 44,
  coinType: 0,
  account: 0,
  change: 0,
  addressIndex: 0
});
console.log(receivingPath); // "m/44'/0'/0'/0/0"

// Generate change address path
const changePath = generateDerivationPath({
  change: 1,
  addressIndex: 5
});
console.log(changePath); // "m/44'/0'/0'/1/5"

// Generate testnet address path
const testnetPath = generateDerivationPath({
  coinType: BIP44_CONSTANTS.COIN_TYPES.BITCOIN_TESTNET,
  addressIndex: 10
});
console.log(testnetPath); // "m/44'/1'/0'/0/10"
