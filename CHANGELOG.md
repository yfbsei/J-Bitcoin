# Changelog

## [2.0.0] - 2025-01-01

### Breaking Changes
- Removed BSV (Bitcoin SV) support
- Removed BCASH (Bitcoin Cash) support
- Now focused exclusively on BTC (Bitcoin)

### Added
- Threshold Signature Scheme (TSS) support
- Hierarchical Deterministic (HD) wallet support
- BIP32, BIP39, BIP44, BIP84 implementations
- Custodial and Non-custodial wallet options

---

## [1.0.2] - Previous (Deprecated)
- Supported BTC, BSV, and BCASH
- **This version is deprecated.** Please upgrade to v2.0.0+
